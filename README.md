# crm-TI

CRM para auxiliar no gerenciamento do setor de TI.

Documentação detalhada por pasta: veja [`DOCUMENTACAO.md`](DOCUMENTACAO.md) na raiz e o mesmo arquivo em cada subdiretório.

**Ambiente:** copie `.env.exemple` para `.env`. Desenvolvimento: SQLite. VPS/produção: PostgreSQL — veja `.vps/DOCUMENTACAO.md` e `pip install -r requirements.txt`.
